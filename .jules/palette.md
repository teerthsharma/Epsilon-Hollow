## 2024-06-14 - Interactive Element Disabled States
**Learning:** Native `title` attributes act as effective, zero-dependency tooltips for disabled buttons to explain state context (e.g. why the button is disabled). Combining `disabled:cursor-not-allowed` and `focus-visible:ring-2` provides comprehensive visual and keyboard feedback.
**Action:** When working on form or interactive elements, always pair disabled states with a clear explanatory title and visual/keyboard interaction cues.
