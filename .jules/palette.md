## 2024-06-07 - Dynamic Titles and Disabled States
**Learning:** Adding dynamic `title` attributes that explain *why* a button is disabled (e.g., "Enter a message to send"), combined with `disabled:cursor-not-allowed` and keyboard accessibility via `focus-visible`, drastically improves context for both mouse and keyboard users.
**Action:** Pair disabled states with dynamic explanatory titles and `focus-visible` styles across all interactive elements.
