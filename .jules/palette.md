## 2024-05-23 - Strict Focus Visible Rings Enhance Chat Accessibility
**Learning:** General `focus:outline-none` combined with custom border colors can fail to provide adequate keyboard navigation feedback, especially on dark mode interfaces. Users need distinct visual cues for element state.
**Action:** Default to `focus-visible:ring-2` coupled with a high-contrast brand color (like `focus-visible:ring-green-500`) for all interactive elements to ensure accessibility standards are met without compromising mouse-click aesthetics.
